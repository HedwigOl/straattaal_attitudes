# Human pretests

This folder contains files for stimuli selection and creation and the data analysis file in **R** for both the stimuli and the label pretest.

---
## Contents

- `Analysis Labels`              - Files used for analysis of the results of the second pretest, testing labels for language varieties and name groups.
- `Analysis Stimuli`             - Files used for analysis of the results of the first pretest, testing recognition and usage of Straattaal words and name associations. Additionally, the two python files used for creating pseudowords and selection the most popular names from the Voornamenbank are included.
