# Human pretests

This folder contains the two **python** files for stimuli selection and creation and the data analysis file in **R** for both the stimuli and the label pretest.

---
## Contents

- `select_popular_names.py`      - Select the most popular names for each decennium
- `pseudowords_creation.py`      - Create pseudowords using Wuggy
- `Analysis Labels`              - Files used for analysis of the results of the second pretest, testing labels for language varieties and name groups.
- `Analysis Stimuli`             - Files used for analysis of the results of the first pretest, testing recognition and usage of Straattaal words and name associations.
